URL Shortener
design a url shortener service that takes in valid URL and returns a shortened url, 
redirecting the user to the previously provided url

Also keep track of total visits/clicks on the URL

Routes
POST/URL - generates a new short URL and returns the shortened url in the format example.com/clicks
GET/:id - redirects the user to the originial url
GET / URL / analytics / :id - returns the clicks for the provided short id.
